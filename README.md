# Online Job Portal 

A simple Online Job Portal built using Java (Servlets + JSP), MySQL, HTML, CSS, and JavaScript.  
Users can browse and apply for jobs, while recruiters can post and manage job listings.

## Features
- User registration & login
- Browse and search jobs
- Apply for jobs
- Recruiter login
- Post and manage job vacancies
- MySQL database support

## Technologies
- Java Servlets & JSP
- JDBC
- MySQL
- HTML, CSS, JS
- Apache Tomcat

## Project Structure
backend/ (Java code + servlets)  
frontend/ (HTML, CSS, JS pages)  
database/ (SQL file)

## Setup
1. Import the project into Eclipse/IntelliJ.  
2. Import `jobportal.sql` into MySQL.  
3. Update DB credentials in `db_connection.java`.  
4. Run the project on Tomcat server.
